# NumFlow 历史文档目录

本目录存放**历史快照**与**阶段性产物**，用于追溯背景，不再作为当前真相。

## 包含内容

- 各轮优化草案：`numflow优化文档*.md`
- 回归 / 实机 / 全链路测试报告
- 前端修复日志
- 已完成或已失效的待办、续篇、迁移说明
- 时间快照型统计报告（如 AI 反馈汇总）

## 使用原则

1. 想了解**当前规则、当前计划、当前部署方式**时，不要先读这里；请先回到上一级 [`../README.md`](../README.md)。
2. 这里的文档允许保留当时语境与结论，因此可能与当前代码状态不一致。
3. 若历史文档中的链接因迁移失效，应优先修成可追溯链接，但**不把历史文档重新包装成当前真相**。
