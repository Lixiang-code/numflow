## 历史文档处理原则

以下内容**不再作为当前真相**，统一放入 [`./history/`](./history/README.md)：

- 回归测试报告、实机测试记录、修复日志
- 多轮 `numflow优化文档*` 这类阶段性草案
- 已完成或已被后续实现覆盖的待办/续篇/迁移说明
- 时间快照型汇总（例如 AI 反馈统计）

如果当前文档与历史文档冲突，**以主目录文档和当前代码实现为准**。
